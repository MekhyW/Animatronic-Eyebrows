# Animatronic-Eyebrows

Small code for controlling two MG90S servo-motors from serial string inputs, used in my fursuit project to create moving eyebrows with multiple expressions. 

CAROTID: C++ code applied to an Arduino Nano board.

Eyebrows.mp4: Demonstration video of the eyebrow system, reacting to inputs in real-time.
